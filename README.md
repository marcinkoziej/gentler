# Gentle - a gentle instagram bot

# Requirements

```
yarn install
```

You need to create a `src/config.ml` file with instagram username and password:
```
let username = "lurker"
let password = "verysecret"
```

# Build
```
npm run build
```

# Watch

```
npm run watch
```

# Run
```
node src/main.bs.js
```


# Editor
If you use `vscode`, Press `Windows + Shift + B` it will build automatically
